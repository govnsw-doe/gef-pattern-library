Call Out Box

The call out box is useful for directing the user’s attention to key content and tasks.

Available in: Squiz Matrix

Usage

  • Keep call-out box content succinct, with clear actions for users to take.
  • Avoid using multiple call-out boxes on a page. This helps retain the box's visual impact.
  • Place the call-out box towards the end of the content page so users are not directed to other links prematurely.

Breadcrumbs

Breadcrumbs reveal the location of the current page within the website’s overall information architecture.

Available in: Squiz Matrix

  • Use enhanced breadcrumbs for large websites with many levels in the information architecture.
  • Enhanced breadcrumbs automatically collapse when a page is more than 4 levels deep.
  • Collapsed links can be accessed by hovering over the … link.

Pug

//- Added: 1.0.0
//- @param {int}     num      - number of links to display
//- @param {boolean} enhanced - added JS functionality to force breadcrumbs on one line
+gef_breadcrumbs(num, enhanced)

Usage

  • The current page always appears as the last item in the breadcrumb and should not be linked.
  • On small screens, both variations of breadcrumbs change to a single link to the parent page.

Accessibility

  • A hidden title announces the proceeding breadcrumb navigation.
  • TAB to access each breadcrumb link in order.
  • Enhanced breadcrumbs:
    • Collapsed links are always accessible with the TAB key.
    • Collapsed links can be expanded by placing keyboard focus on …, then hitting ENTER.

Local footer

The local footer shows key links and social profiles for a program, team, or business unit.

Available in: Squiz Matrix

The local footer works in conjunction with the context set from the section homepage.

Example

Code

//- Added: 1.0.0
//- @param {string} channels      - comma delimeted list of social media channels.
//- Available channels: "Yammer, Twitter, Facebook, YouTube, Instagram, Google Plus"
//- @param {bool} show_link_list  - choose to display the link list
+gef_section_footer(channels, show_link_list)

Usage

  • The types of links can include:
    • Occasional content, eg “about your team”
    • Regular updates, eg “news”
    • Key actions, eg “contact”
  • Each link title should be short and concise.
  • Add up to 5 links to the local footer.
  • These links can point to other pages in the section or to orphan pages.
  • Available social profile links:
    • Yammer
    • Facebook
    • Twitter
    • YouTube
    • Instagram
    • Google Plus
  • Do not use the same social profile more than once in the local footer.
  • Squiz Matrix users:
    • The social profile links can only be configured by platform administrators.

Accessibility

  • When a social profile icon receives keyboard focus, its name should be announced to the screen reader.

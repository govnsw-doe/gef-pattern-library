Anchor box

Anchor boxes help users ascertain a page’s content and quickly navigate to specific blocks on the page.

Available in: Squiz Matrix

Example

On this page

    Code

    Using pug

    @param {int} num - number of links

    +gef_anchor_box

    Usage

    • Anchor boxes are dynamically generated based on the H2 headings on the page. This makes them particularly useful for navigating long content pages.
    • Be aware of the number of H2 headings on the content page as they will be automatically listed in the anchor box.
    • Use the anchor box as a navigation tool for short FAQ content. You can do this by presenting each question as an H2 heading.
    • Squiz Matrix users: anchor boxes automatically appear on standard content pages that contain three or more H2 headings. You can choose to disable them.

    Accessibility notes

    • Use TAB key to enter the anchor box.
    • TAB to move through the list of links.
    • ENTER to select a link. This will cause the page to jump to the related heading. The next TAB should focus on content under the heading.

Link groups

Link groups promote useful links to a section’s audience groups.

Available in: Squiz Matrix

Link groups provide a way to reveal and highlight key content from a section’s information architecture so that likely audience groups can find them more easily.

Example

  • Nam quis nulla

    • Phasellus rhoncus aenean id metus id.
    • Nulla non lectus sed nisl molestie.
    • Maecenas sollicitudin nullam rhoncus aliquam metus..

  • Nullam feugiat turpis

    • Fusce nibh, curabitur sagittis hendrerit ante..
    • Duis condimentum augue id magna semper.
    • Fusce consectetuer risus a nunc. Aliquam.

  • Morbi gravida libero

    • Proin mattis lacinia justo. Vestibulum facilisis.
    • Integer rutrum, orci vestibulum ullamcorper ultricies.
    • Aenean vel massa quis mauris vehicula.

Code

Using pug

+gef_linkgroup_list(3,1)
@param {int} num  - number of linkgroups to display. Valid inputs: 1, 2, 3, 4
@param {int} links - number of linkgroups items within a linkgroup to display.

Usage

  • Choose an appropriate title for the link group to describe the type of links it contains.
  • Lists in a link group should form a unified whole, ie do not mix different types of lists.
  • A link group can support 3 or 4 lists of links.
  • Each list has a title, generally the audience name.
  • Each list can contain up to 5 links.
  • Suggested link groups: popular information, key tasks, common systems.
  • Squiz Matrix users:
    • Link groups are only available on the section homepage template.

Code

Using pug

@param {max} - Choose the maximum amount of tabs to be displayed
+gef_tabs(4)

Usage

  • The title for each tab should be short (up to 25 characters).
  • Titles should be descriptive of the content contained in the tab.
  • A group of tabs should contain content that highly relate to one another.
  • In small screens, tabs will appear as an accordion.
  • Squiz Matrix users:
    • Each group of tabs can contain a maximum of 4 tabs.

Accessibility

  • Use TAB key to enter a group of tabs.
  • TAB to move through each tab’s title.
  • ENTER to select a tab.
  • After a tab is selected, TAB to focus on its contents.

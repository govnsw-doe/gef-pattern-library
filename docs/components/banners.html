Banners

The banner gives publishers a way to personalise their section homepage with a key message and background image.

Available in: Squiz Matrix

Default banner

Featured page

Nam quis nulla..

Phasellus rhoncus aenean id metus id velit ullamcorper pulvinar. Vestibulum fermentum tortor id mi.

Nulla 
+gef_banner(&quot;//placehold.it/1350x520&quot;)
  • Keep the title and description text short so they fit inside the key message box.
  • The key message button should link to a relevant page within the current section.
  • The default banner background image should be 1350 px × 520 px.

Short banner

Automatically hidden on mobile

Featured page

+gef_banner(&quot;//placehold.it/1350x250&quot;, true)
  • The short banner background image should be 1350 px × 250 px.

Code

Pug

//- Added: 1.0.0 / Updated: 1.0.1
//- @param {object} image_src - banner image width and height
//- @param {boolean} small_banner - True if small banner otherwise big banner
+gef_banner(image_src, small_banner)

Usage

  • There are two types of banners:
    • Default: a taller image banner with space for a key message.
    • Short: a shorter, image-only banner.
  • Squiz Matrix users:
    • Banners are only available on the section homepage template.

Accessibility

  • Do not forget to add appropriate alt text to describe the background image.

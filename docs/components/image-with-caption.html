Image with caption

A caption describes its associated image so users get a better understanding of its contents and purpose.

Available in: Squiz Matrix

Nam quis nulla. Integer malesuada, in enim a arcu imperdiet malesuada. 

mixin Image
  img(src="http://placehold.it/800x400" alt="Placeholder" title="Placeholder image")
+gef_image_with_caption

Code

Pug

//- Added: 1.0.0
//- @mixin Image - use this mixin for your image tag
mixin Image
  img(src="http://placehold.it/800x400" alt="Placeholder" title="Placeholder image")
+gef_image_with_caption

Usage

  • Images should only be used when they add value and/or provide context to the content (not solely for decoration).
  • Any image added to a template’s main content body should always include a caption.
  • Captions should be useful and succinct (up to 150 characters).
  • Captions should give additional background and context.
  • Captions for charts or graphs should include a figure number and label text. Example:
    • “Figure 1: Components of the Resource Allocation Model explained”
  • You do not need to use a caption if your image is a screenshot that’s described by the text that surrounds it.

Accessibility

  • Do not forget to add appropriate alt text to the image.
  • The alt text should describe the image contents literally.

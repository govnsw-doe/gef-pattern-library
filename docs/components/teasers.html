Teasers

Teasers are used to visually highlight a set of noteworthy content pages.

Available in: Squiz Matrix

Example

  • Nam quis nulla

    Phasellus rhoncus aenean id metus id vel.

    Vestibulum fermentum tortor id mi.

  • Nulla non

    Vivamus luctus egestas leo..

    Maecenas sollicitudin nullam rhoncus aliquam metus.

  • Etiam egestas wisi

    Aliquam erat volutpat..

    Nunc auctor, Mauris pretium quam et urna.

Code

Using pug

+gef_teaser_list(3)
@param {int} num  - number of teasers to display. Valid inputs: 1, 2, 3, 4

Usage

  • Teasers are made up of links to 3 content pages.
  • Each link includes:
    • Label: provides additional context around the link.
    • Title: the title of the linked page.
    • Thumbnail: a thumbnail that visually describes the content.
    • Description: a short description about the content.
  • The types of content appropriate for teasers are: popular, useful, important, timely, or newsworthy.
  • Squiz Matrix users:
    • Teasers are only available on the section homepage template.
    • The labels are selectable from a predefined list of options.
    • The title, thumbnail, and description are automatically pulled from the linked page.

Accessibility

  • Each thumbnail should include an alt text to describe the image.
